## Install

```bash

```

`@booklib/skills` continues to receive updates for backwards compatibility, but `@booklib/core` is the primary package going forward.

- Knowledge graph with capture and graph-augmented search
- Interactive setup wizard
- Doctor diagnostics with `--cure`
